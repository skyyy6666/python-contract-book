# 通讯录管理系统 (Contact Book System)

一个基于Python开发的命令行通讯录管理系统，具备完整的CRUD功能和异常处理。

## ✨ 功能特性

- 📝 **添加联系人** - 保存姓名和电话到本地文件
- 📋 **浏览联系人** - 查看所有已保存的联系人
- 🔍 **查找联系人** - 按姓名搜索特定联系人
- ⚡ **异常处理** - 完善的输入验证和错误处理
- 💾 **数据持久化** - 所有数据本地文件存储

## 🛠 技术栈

- Python 3
- 文件IO操作
- 异常处理机制
- Git版本控制

## 🚀 快速开始

### 环境要求
- Python 3.6+

### 安装使用
1. 克隆项目
```bash
git clone https://github.com/skyyy6666/shiny-tribble.git
```
2. 运行程序
```bash
cd shiny-tribble
python 通讯录.py
```
